
BGP DUAL-HOMED — CONFIGURACIÓN DE ROUTERS
Topología: TELVISO (AS100) — SRT (AS65000) — METROTEL (AS200)  |  Packet Tracer Lab
  TELVISO  —  AS 100  |  ISP PRIMARIO

Datos de interfaz
Loopback0: 100.100.100.1 / 24  (red anunciada por BGP)
GigabitEthernet0/0: 10.0.0.1 / 30  (enlace hacia SRT)
BGP
AS Number: 100
Vecino: 10.0.0.2  →  SRT (AS 65000)
Network anunciada: 100.100.100.0 / 24
Ruta estática: 192.168.1.0/24  vía  10.0.0.2
Configuración completa
hostname TELVISO
!
interface Loopback0
 ip address 100.100.100.1 255.255.255.0
!
interface GigabitEthernet0/0
 ip address 10.0.0.1 255.255.255.252
 duplex auto
 speed auto
!
router bgp 100
 bgp log-neighbor-changes
 no synchronization
 neighbor 10.0.0.2 remote-as 65000
 network 100.100.100.0 mask 255.255.255.0
!
ip classless
ip route 192.168.1.0 255.255.255.0 10.0.0.2

  SRT  —  AS 65000  |  ROUTER CORPORATIVO (DUAL-HOMED)

Datos de interfaz
GigabitEthernet0/0: 10.0.0.2 / 30  (hacia TELVISO — NAT outside)
GigabitEthernet0/1: 20.0.0.2 / 30  (hacia METROTEL — NAT outside)
GigabitEthernet0/2: 192.168.1.1 / 24  (LAN interna — NAT inside)
BGP
AS Number: 65000
Vecino 1: 10.0.0.1  →  TELVISO (AS 100)
Vecino 2: 20.0.0.1  →  METROTEL (AS 200)
Network anunciada: 192.168.1.0 / 24
NAT y rutas
NAT Overload (PAT): LAN → GigabitEthernet0/0 (ISP primario)
Default route primaria: 0.0.0.0/0  vía  10.0.0.1  (TELVISO)
Default route secundaria: 0.0.0.0/0  vía  20.0.0.1  métrica 10  (failover)
Configuración completa
hostname SRT
!
interface GigabitEthernet0/0
 ip address 10.0.0.2 255.255.255.252
 ip nat outside
!
interface GigabitEthernet0/1
 ip address 20.0.0.2 255.255.255.252
 ip nat outside
!
interface GigabitEthernet0/2
 ip address 192.168.1.1 255.255.255.0
 ip nat inside
!
router bgp 65000
 bgp log-neighbor-changes
 no synchronization
 neighbor 10.0.0.1 remote-as 100
 neighbor 20.0.0.1 remote-as 200
 network 192.168.1.0
!
ip nat inside source list Local interface GigabitEthernet0/0 overload
ip classless
ip route 0.0.0.0 0.0.0.0 10.0.0.1
ip route 0.0.0.0 0.0.0.0 20.0.0.1 10
!
ip access-list standard Local
 permit 192.168.1.0 0.0.0.255
